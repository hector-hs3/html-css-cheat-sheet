HTML5 & CSS3 Cheat Sheet

HTML Structure

HTML5 uses the following hierarchical structure:

<!DOCTYPE html>
  Identifies language of file
<html>
  Denotes start of code
<head>
  Metadata of the file (content is not visible in browser)
<title>
  Title of the site (appears in the tab on browser)
<link/>
  Used to link other files (like css or frameworks) or Content Management Systems
</head>

<body>
 Visible content of website

<header>
  Top section of a website, usually contains headings (h1, h2, h3, etc...), navigation/list (nav, ul, li) elements.
</header>

<main>
  Middle section of a website that holds main content. Tags used in this section include:
  • <section>
  • <article>
  • <aside>
  • <figure>
  • <figcaption>
  • <audio>
  • <source>
  • <video>
</main>

<footer>
  Bottom section of a website that usually displays contact infro, copy-rights, and other information of the such.
</footer>

</body>

</html>
