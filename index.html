HTML Structure

HTML5 follows the follow hierarchical structure:

<!DOCTYPE html>  <= Identifies language of file
<html>  <= Denotes start of code
<head>  <= Metadata of the file (content is not visible in browser)
<title>  <= Title of the site (appears in the tab on browser)
<link/>  <= Used to link other files or Content Management Systems
</head>
<body>  <= Visible content of website
</body>
